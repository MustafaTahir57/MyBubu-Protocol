# MYBUBU Ecosystem — Complete User Flow

---

## Phase 1 — Entry & Investment

### Step 1: Connect Wallet
- User clicks "Connect Wallet" → MetaMask or WalletConnect modal opens
- Must be on BSC network (Mainnet/Testnet based on config)
- If wrong network → auto-switch prompt + "Wrong Network" banner appears
- Wallet address displayed in header as `0x1234...abcd`

### Step 2: Buy MyBoo Token (Presale)
- **Purpose:** Early investment before official launch
- Two payment options:
  - **BNB** — Min 0.1 BNB, Max 2 BNB → calls `buyWithBNB()` (sends BNB as value)
  - **USDT** — Min 100 USDT, Max 10,000 USDT → requires USDT approval first, then calls `buyWithUSDT(amount)`
- **Rate:** 10,000 MyBoo per 500 USDT equivalent
- Real-time balance validation before transaction
- Button states: `Wallet Opening...` → `Approving...` (USDT only) → `Purchasing...` → ✅ Success toast

---

## Phase 2 — Token Migration

### Step 3: Swap MyBoo → MYBUBU (1:1)
- User's MyBoo balance shown (fetched from presale contract `userInfo`)
- Enter amount → quick select buttons (25%, 50%, 75%, 100%)
- Two-step transaction:
  1. **Approve** MyBoo tokens to Swap contract → `approve(swapAddress, amount)`
  2. **Swap** → `swap(amount)` — burns MyBoo, mints MYBUBU to wallet
- **Ratio:** 1 MyBoo = 1 MYBUBU (zero fees)
- After swap, user now holds MYBUBU tokens

---

## Phase 3 — Ecosystem Onboarding

### Step 4: Join via Upline
- Requires **1 MYBUBU** token (transferred to upline as join fee)
- Upline address auto-populated from referral URL (`/ref/0xABC...`) or defaults to `0x000...`
- Two-step transaction:
  1. **Approve** 1 MYBUBU to presale contract
  2. **Transfer** 1 MYBUBU to upline address
- Unlocks: referral link generation, 10-level reward structure
- Referral format: `https://mybubu.io/ref/{userAddress}`
- "Invite a Friend" section with copy-to-clipboard

---

## Phase 4 — Earning (Post-Join)

### Step 5: MyMomo — Vested Staking
- Deposit MYBUBU → receive MyMomo (vested position)
- Requires MYBUBU approval → then `deposit(amount)`
- Rewards vest over time
- Claim vested MYBUBU rewards via `claim()`

### Step 6: Deposit BNB — LP Mechanism
- Deposit BNB into liquidity pool mechanism
- Earn claimable BNB rewards over time
- Claim via `claimDividends()`

### Step 7: Mint NFT Nodes
- Purchase NFT Nodes using USDT (approve → mint)
- NFT holders earn **dual rewards:**
  - 🪙 **MYBUBU token rewards** — claim via `claimTokenRewards()`
  - 💎 **BNB dividends** — claim via `claimDividends()` (from NFT_NODE contract)
- View pending rewards in real-time on dashboard
